## Core formula

1. **Episode topic = one historical event** (not a generic “visit to Belém”).
2. Gaivota flies in, **witnesses** the event (or its echo / memory / street retelling tied to that event).
3. Reader learns **Portuguese history + culture** through the story; vocab is support, not the product pitch.
4. Standalone: any episode can be bought alone.

Pitch line:

> A noir seagull over Portugal. Each issue: one real historical event. 5 pages. 100 new Portuguese words. HTML first; PDF later.

### History brief → comic (locked)

| Sheet | Role |
|--------|------|
| **1. History brief** | Short factual summary + **one textbook-style history image**. Clear, school-accurate. **Not** the comic. |
| **2–6. Comic (5 pages)** | **Artistic interpretation** through Gaivota’s eyes — noir B&W cartoon. May compress time, invent composite characters, dramatize. |
| **Study** | Vocab + comprehension + cultural note |

Rule for the brief page:

- ~120–200 words EN (or bilingual labels: dates/names in PT, prose in EN for v1)
- timeline bullets OK (3–5 facts)
- one line: *“The comic that follows is an artistic interpretation.”*
- no speech bubbles on this page
- **Hero image: history-textbook style** (see below) — visually distinct from the comic

### Brief-page image (locked)

Looks like a **school history textbook / atlas plate**, not noir comic art:

| | |
|--|--|
| Look | Engraving, lithograph, archival map, museum plate, or calm didactic illustration |
| Tone | Documentary, neutral, “chapter opener in a history book” |
| Color | Sepia / muted print / soft grayscale OK — **not** high-contrast noir ink |
| Content | The event itself, a key place, a period map, or a famous motif of that moment |
| Caption | Short textbook caption under the image (EN + date/place) |
| Forbidden on this sheet | Speech bubbles, Gaivota as cartoon hero, screentone action panels |

Contrast rule: reader should feel **“textbook → then comic”** when turning the page.

Rule for the comic:

- does **not** replace the brief as a textbook
- must stay emotionally true to the event, not invent false “facts” that contradict the brief
- if dramatized, the brief remains the source of truth
- art style = noir B&W cartoon (**different** from the brief image)

Avoid: cute pastel travel comic; making the brief page look like another comic page; purple AI gradients.

## Editorial voice (locked)

Two registers. Never mix them on the same sheet.

### A. History brief — *storyteller historian*

Factual, but **not dry**. Readers should want to turn the page into the comic.

| | |
|--|--|
| Persona | A great history teacher / documentary opener — vivid, human, clear |
| Person | Third person; no Gaivota “I” on this sheet |
| Tone | Engaging, concrete, a little cinematic; still accurate |
| Hook | Open with a striking image, stakes, or “why this day mattered” |
| Length | Short paragraphs; sensory details OK (*smoke, bells, silence*) if true to sources |
| EN | Clear adult English; warm and readable |
| PT on this sheet | Names, places, dates, event titles |
| Avoid | Encyclopedia blandness; joke-meme tone; fake quotes; spoiling every beat of the comic; Gaivota POV |

Recipe for the brief (~120–200 words):

1. **Hook** — one vivid sentence (the moment / the stakes)
2. **What happened** — who, when, where (clean facts)
3. **Why it matters** — Portugal / Europe / today
4. **Bridge** — “The comic that follows is an artistic interpretation.”

Example vibe:

> At All Saints’ Day mass in 1755, Lisbon’s churches were full — and then the ground failed. In minutes, quake, fire, and a wall of water remade one of Europe’s great capitals. Thousands died; the shock reached philosophers across the continent. Portugal would rebuild Lisbon on a new plan — and never forget that morning.  
> *The comic that follows is an artistic interpretation.*

### B. Comic — *Gaivota (witness)*

| | |
|--|--|
| Persona | A seagull over Portugal; sees, does not lecture |
| Person | First person asides, short (*Eu vi…* / *Olhem…*) |
| Tone | Noir, dry, slightly poetic; adult cartoon |
| Length | 1–2 lines per page max for Gaivota |
| Dialogue (humans) | PT-PT, natural for the era’s *feel*, but readable A2–B1 |
| Avoid | Cute mascot voice; explaining the whole history in bubbles; breaking the fourth wall with “learn this word” |

Example vibe:

> Gaivota: O chão tremeu. A cidade gritou. Eu voei mais alto.

### C. Study page — *teacher footnote*

- Warm, practical, short.
- Glossary = dictionary-clean (PT → EN).
- Cultural note = one sticky insight, same factual register as the brief (not noir).
- Comprehension Qs = clear exam-style prompts.

### House rules

1. Brief makes you **care what happened**. Comic shows **how it felt**.
2. Gaivota never contradicts the brief.
3. Curiosity and dignity over propaganda or nationalist chest-beating.
4. Sensitive episodes (dictatorship, war, 25 de Abril): human-scale; brief stays vivid but fair; comic avoids cartoon cruelty.
5. If a sentence is accurate but boring — rewrite it.

### Comic = a complete story (locked)

Each episode’s 5 comic pages must be a **real comic story**: beginning → middle → end. Not a mood board, not a slideshow of history facts, not an open fragment.

| Beat | Page role |
|------|-----------|
| **Setup** | World + who we follow + what is at stake |
| **Rising action** | Event starts; pressure builds |
| **Turning point** | The decisive moment of *this* story |
| **Falling action** | Immediate consequence |
| **Resolution** | Emotional or narrative **closure** — the issue feels finished |

Hard rules:

- There is a **through-line character or pair** the reader follows (plus Gaivota as witness).
- Page 5 **lands**: decision made, fate clear, silence after the storm, dawn after the night — something that *ends*.
- No “to be continued” for the episode’s own plot (season arc can echo later; this issue closes).
- History can be bigger than 5 pages; the **comic story** is a complete human-scale slice of that event.

Suggested page map:

1. Setup / omen (Gaivota arrives; introduce people + stake)
2. Rising action
3. Turning point
4. Consequence
5. **Resolution** + Gaivota leaves (or watches the last quiet frame)

## Production pipeline (per episode)

### Now (HTML)

0. **History brief** — factual sheet (source of truth for the episode)
1. **Vocab lock** — 100 new + recycle list
2. **Script** — PT-PT noir dialogue + Gaivota asides (artistic interpretation)
3. **Panel board** — 5 comic pages, B&W notes
4. **Art (generate once)** — (a) textbook brief image · (b) cover · (c) 5 comic pages · (d) mascot already locked in `brand/`
5. **HTML layout** — cover + history brief (+ textbook image) + 5 comic + study
6. **Study pack** — glossary + Qs + cultural note (in HTML)
7. **QA** — brief vs comic consistency + PT-PT pass + vocab count = 100

### Art lock rule (LOCKED)

**Generate each episode’s images once in the pipeline. Do not regenerate on every rewrite.**

| | |
|--|--|
| First pass | Create art → save under `episodes/NN-…/art/` (and `brand/` for mascot) |
| Later edits | Change **script, bubbles, HTML, vocab, brief copy** — keep the **same** image files |
| Regenerate only if | User explicitly says to replace a specific file (e.g. “redo p3 art”) |
| Forbidden | Re-rolling the whole episode art because dialogue or layout changed |

Why: consistency of character/scene, cost/time, and a stable visual “first generation” as the canon for that episode.

If a page needs a new image, replace **that filename only** and note it in the episode `BRIEF.md` (*art revision: p3*).

9. **Gumroad variant** — $5, cover, description
